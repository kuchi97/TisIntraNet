  - 주제 선정
  GUI구현부터 DB운용까지 효과적으로 학습하며 설계할 수 있다고 생각되는 가상의 기업체 내 인트라넷 웹페이지를 개발 모델로 선정
  
  - 프로젝트 목표
  1. 로그인부터 페이지 내 메뉴 사용까지 사원DB정보를 기반으로 페이지 운용

  3. DB안의 자료를 이용하여 자유로운 입출력이 가능한 게시판 구현
  4. 사원의 쪽지함을 개별 DB로 설계하여 각각의 쪽지함 구현
